# SwiftScout 🕵️♂️ - Xcode Source Editor Extension

![Swift Version](https://img.shields.io/badge/Swift-5.5-orange)
![Xcode Version](https://img.shields.io/badge/Xcode-13%2B-blue)

Eine intelligente Xcode-Erweiterung zur visuellen Analyse von Funktionsbeziehungen im Code.

## 🚀 Features
- **Globaler Event-Listener** für Tastatur/Maus-Events
- **Shift+Cmd+Klick** zur Funktionsanalyse
- **Regex-basierte** Funktionsnamensextraktion
- **SourceKit-LSP Integration** für präzise Code-Analyse
- **Dynamisches Popup-UI** mit Dateivorkommen
- Echtzeit-Aktualisierung der Ergebnisse

## 📦 Installation
1. Projekt klonen:
```bash
git clone https://github.com/NEO849/SwiftScout.git
   ```
2. open SwiftScout.xcodeproj
